# XCrawl Hugging Face Space

Demo Streamlit app for XCrawl Web Scraping Proxy.

## Deploy

1. Fork this folder to a GitHub repo
2. Go to https://huggingface.co/new-space
3. Connect GitHub repo
4. Space SDK: Streamlit
5. Deploy!

## Usage

- Open the Space URL
- Enter your XCrawl API key
- Enter a URL
- See results in real time

## Purpose

- **Demo**: Show potential customers how XCrawl works
- **Lead gen**: Space visitors → XCrawl signups
- **Portfolio**: For Upwork/Fiverr proposals
